# 竞赛老师反馈 - 功能与页面实现清单

> 根据竞赛老师建议，梳理需要新增及修改的具体页面和功能模块。

---

## 一、渐进式技能评分体系（双维度）

**建议核心**：AI 分析社交媒体 + 社区持续评价，综合给技能提供者打分。

### 1.1 AI 分析社交媒体（新增模块）

| 功能点 | 实现位置 | 说明 |
|--------|----------|------|
| **平台授权入口** | 个人中心 / 技能发布页 | 授权 B站、小红书、公众号、YouTube 等 |
| **AI 分析结果展示** | 技能详情页 | 展示「AI 可信度评分」及来源说明 |
| **评分维度说明** | 技能详情页 / 帮助页 | 说明：持续输出 > 播放量/粉丝数 |

**需新增/修改的页面：**

- **个人中心**：新增「社交账号授权」区块，可连接 B站、小红书、公众号、YouTube
- **技能详情页**：新增「AI 可信度评分」卡片 + 来源说明（如「基于 B站 12 篇持续分享」）
- **技能发布/编辑页**：在发布技能时引导用户选择授权平台

---

### 1.2 社区内互相反馈 + 评价（修改 + 新增）

#### 2.1 公开交换反馈帖子区

| 功能点 | 实现位置 | 说明 |
|--------|----------|------|
| **自动创建反馈帖** | 交换确认页 / 流程 | 双方确认交换后，自动创建「交换反馈帖」 |
| **交换反馈帖列表** | 新增页面/板块 | 展示所有交换反馈帖，可筛选 |
| **学习进度记录** | 交换反馈帖详情 | 双方可记录学习进度、心得、反馈 |
| **互动活跃度展示** | 技能详情页 / 个人中心 | 展示「反馈次数」「互动活跃度」等 |

**需新增的页面：**

- **交换反馈帖列表**：可在「发现页-社区」或「我的主页」下新增 Tab
- **交换反馈帖详情**：类似小红书帖子详情，支持进度、评论、点赞
- **交换确认流程**：在确认交换时增加「自动创建反馈帖」步骤

**需修改的页面：**

- **发现页-社区**：增加「交换反馈」类帖子或筛选
- **技能详情页**：增加「交换反馈」相关数据展示

---

#### 2.2 私聊板块 Soul 式成就（类似 Soulmate）

| 功能点 | 实现位置 | 说明 |
|--------|----------|------|
| **交换阶段里程碑** | 消息页-聊天详情 | 展示「交换进度」阶段（如 1/5 完成） |
| **点亮成就** | 消息页-聊天详情 | 达到一定阶段解锁「点亮」图标/成就 |
| **社交互动记录** | 消息页 | 聊天、视频、交换次数等可被记录并展示 |

**需修改的页面：**

- **消息页**：在聊天头部或侧边栏增加「交换进度」卡片
- **聊天详情**：增加「阶段成就」展示（如「首次交换」「达成 5 次会话」等）

**需新增的组件：**

- **交换阶段进度条**：展示当前交换阶段（1/5、2/5…）
- **成就点亮动画**：达到某个阶段时点亮对应图标

---

#### 2.3 NFT 成就集卡

| 功能点 | 实现位置 | 说明 |
|--------|----------|------|
| **NFT 生成** | 后端 + 触发逻辑 | 交换完成后，AI 根据互动数据生成 NFT |
| **贡献分配** | NFT 详情 | 展示双方贡献占比（如 60% / 40%） |
| **NFT 橱窗** | 个人中心 | 展示该用户所有交换 NFT |
| **NFT 详情** | 新增页面/弹窗 | 展示交换故事、贡献、时间线 |

**需新增的页面：**

- **个人中心**：新增「NFT 橱窗」区块，展示所有交换 NFT
- **NFT 详情页/弹窗**：点击 NFT 查看交换故事、双方贡献、时间线
- **技能提供者 NFT 展示**：在技能详情页可查看该技能提供者的 NFT 橱窗

**需修改的页面：**

- **个人中心**：在成就墙中增加「NFT 展示」入口
- **技能详情页**：增加「查看 TA 的交换 NFT」入口

## 三、推荐实现优先级（P0 / P1 / P2）

### 分级原则

- **P0**：竞赛展示核心，可 mock 演示，直接对应老师反馈要点
- **P1**：加分项，可 UI 先行，逻辑简单
- **P2**：差异化亮点，实现成本高，可后续迭代

---

### P0 - 高优先级（必须展示）

| 功能 | 具体内容 | 理由 |
|------|----------|------|
| **交换反馈帖** | 交换反馈帖列表、详情、自动创建流程 | 老师明确提到「平台自动触发开启交换反馈帖子区」，是社区反馈的核心 |
| **消息页交换进度** | 交换阶段进度条（如 1/5）、阶段成就展示 | 对应「线上社交被记录」「Soulmate 式点亮」，易 mock 且效果直观 |
| **消息页成就点亮** | 达到阶段时点亮图标/成就 | 增强互动乐趣，展示成本低 |

**实现要点**：Mock 数据即可，重点展示流程和 UI。

---

### P1 - 中优先级（建议展示）

| 功能 | 具体内容 | 理由 |
|------|----------|------|
| **AI 可信度评分** | 技能详情页增加「AI 可信度」卡片 + 来源说明 | 展示双维度评分中的「AI 分析」维度，用 mock 分数即可 |
| **社交账号授权** | 个人中心/设置入口，展示「连接 B站、小红书」等按钮 | 体现「授权平台」概念，无需真实 OAuth |
| **成就墙 + NFT 橱窗** | 个人中心：成就墙（徽章）与 NFT 橱窗并列 | 展示「成就集卡」概念，NFT 橱窗可先用 mock 卡片 |

**实现要点**：以 UI 和文案为主，逻辑可简化。

---

### P2 - 后续迭代（可后续）

| 功能 | 具体内容 | 理由 |
|------|----------|------|
| **NFT 生成与贡献分配** | 交换完成后 AI 生成 NFT、贡献占比 | 需要后端逻辑 + 设计，竞赛阶段可先做静态展示 |
| **技能提供者 NFT 橱窗** | 技能详情页查看 TA 的 NFT | 依赖 NFT 数据，P1 橱窗做好后再扩展 |
| **真实社交平台 API** | 对接 B站、小红书等 OAuth + 内容解析 | 工程量大，竞赛外可做 |

---

### 优先级汇总表

| 优先级 | 功能模块 | 页面/组件 |
|--------|----------|-----------|
| **P0** | 交换反馈帖 | 列表、详情、自动创建 |
| **P0** | 消息页 Soul 式成就 | 交换进度、阶段成就、点亮效果 |
| **P1** | AI 可信度评分 | 技能详情页卡片 |
| **P1** | 社交账号授权 | 个人中心/设置入口 |
| **P1** | NFT 橱窗（静态） | 个人中心成就墙旁 |
| **P2** | NFT 完整流程 | 生成、贡献分配、详情 |
| **P2** | 技能提供者 NFT 展示 | 技能详情页入口 |
| **P2** | 真实社交平台对接 | 授权 + 内容解析 |
